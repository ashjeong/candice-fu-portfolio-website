# candice-fu-portfolio-website

Created by [ashjeong](https://github.com/ashjeong) & [magandren](https://github.com/magandren) for [Candice Fu](https://www.linkedin.com/in/candice-fu/).

This project is a re-work of a previously created website with low responsiveness and broken views when viewing on smaller screens. If interested, please view the project intent [here](https://www.figma.com/deck/7PCL14Jfaulk597DOkm6I7/Super-cool-website?node-id=1-42&t=1r8HrmVOaM1gGEZd-1).

## Setup

Install all dependencies:

```bash
npm install
```

## Run

Run up the website

```bash
npm run dev
```
